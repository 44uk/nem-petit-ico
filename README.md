# nem Petit ICO

## Introduction

For personal or small ICO project use.

*Not supported MAINNET now. Do not use on MAINNET.*

## Usage

* [nem Petit ICO](https://44uk.github.io/nem-petit-ico/)

### Setup

```node
$ npm install
```

### Setting

Copy `.env.sample` to `.env` and Edit.

### TokenSale

```node
$ npm run tokensale
```
